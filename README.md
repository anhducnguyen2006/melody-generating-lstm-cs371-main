# Melody Generation with LSTM - 4th Hour Project CS371

### 1. To run Presentation

#### Mac
```bash
open lstm_midi_project.html
```

#### Windows
```bash
start lstm_midi_project.html
```

### 2. To run Web Application
```bash
cd webapp
npm run install:all   # one-time setup (already done)
npm start             # starts both server (3001) and client (3000)
```
